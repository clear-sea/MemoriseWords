
# 记单词小程序

用来帮助你记单词。使用PyQt5开发

## 文件结构

+ images:图片文件夹

+ UI:界面python代码
    + about.ui:“关于”页面的ui文件
    + show.ui:“浏览单词表”页面的ui文件
    + main.ui:主界面的ui文件
    + entryDialog:输入单词对话框
    + passWords:挨个过单词

+ style.qss:主Qt样式表
+ resource.rc:资源文件
