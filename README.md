# 记单词小程序

用来帮助你记单词。使用PyQt5开发

## 文件结构

+ images:图片文件夹
    + add.png:“添加单词”的图标
    + dict.png:“生词本”图标
    + logo.png:程序图标
    + memorize.png:“记单词”图标
    + test.png:“测验”图标

+ about.ui:“关于”页面的ui文件
+ show.ui:“记单词”页面的ui文件
+ main.ui:主界面的ui文件

+ style.qss:主Qt样式表

+ Ui_about.py:“关于”页面编译后得到的py文件
+ Ui_main.py:“记单词”页面编译后得到的py文件
+ Ui_show.py:主页面编译后得到的py文件
